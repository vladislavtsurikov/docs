Advanced Brush is the main interactive painting tool in MegaWorld. It lets you paint directly in the scene view while still using the same shared systems that power the procedural tools.

This makes it a strong bridge between manual art direction and rule-based placement.

<Info>
  New to Advanced Brush? Follow the step-by-step [Advanced Brush — Getting Started](/megaworld/getting-started/advanced-brush) tutorial first.
</Info>

## Features

### The most realistic spawn

Advanced Brush combines manual painting with the full MegaWorld placement stack — filters, transforms, overlap checks, and scatter. Every stroke respects the same rules as procedural passes, so hand-painted areas blend naturally with stamper-generated results.

### Spawn support for all resource types

Advanced Brush works with all four MegaWorld resource kinds:

- [Terrain Object](/megaworld/resource-types/terrain-object) and [Unity/GameObject](/megaworld/resource-types/unity-gameobject) — painting with filters, transforms, and overlap checks
- [Unity/Terrain Detail](/megaworld/resource-types/unity-terrain-detail) — interactive grass and mesh detail painting
- [Unity/Terrain Texture](/megaworld/resource-types/unity-terrain-texture) — direct terrain texture painting with mask filter support

### Shared settings with Terrain Spawner

Types in MegaWorld are reusable presets. A Type configured in [Terrain Spawner](/megaworld/tools/terrain-spawner) can be used directly in Advanced Brush with no extra setup. Build your rules once in the stamper, then hand-paint with the same rules in the brush.

### Live visualization

Advanced Brush draws the active preview in the scene view while you paint:

- With a **Mask Filter** active, you see the filter output overlaid on the terrain.
- With **Simple Filter** selected, you see a handle-based preview of the placement area.

Visualization settings are controlled globally in [Preferences](/megaworld/preferences/overview).

### Spherical spawn support

Enable **Spherical** mode to paint on non-terrain surfaces such as sphere geometry or curved meshes. In spherical mode, Mask Filters are replaced by Simple Filter, and only compatible scatter algorithms are available.

### Cell support

Placed objects are stored in spatial cells. This enables [Overlap Check Settings](/megaworld/overlap-check-settings/overview), and makes tools like [Brush Erase](/megaworld/tools/brush-erase) and [Edit](/megaworld/tools/edit) aware of exactly where each object was placed.

## Tool-level controls

Advanced Brush has two main tool-level settings blocks:

- **Advanced Brush Tool Settings**
- **Brush Settings**

## Advanced Brush Tool Settings

- **Texture Target Strength** controls how strongly terrain textures are painted.
- **Enable Failure Rate On Mouse Drag** lets the drag workflow use an additional failure chance while painting continuously.
- **Failure Rate On Mouse Drag** controls that extra failure chance.

## Brush Settings

Brush Settings control the shape and behavior of the brush itself. Advanced Brush uses the extended brush variant with three subsections.

### Brush Spacing

Controls how frequently brush marks are applied as you drag the mouse.

- **Spacing Mode** — enables or disables Brush Drag. When disabled, the brush only fires on click, not while dragging.
- **Brush Spacing Equals Type** — sets the unit that determines spacing distance (e.g. based on brush size or a fixed value).
- **Spacing Range** — the minimum and maximum distance limits for spacing.
- **Brush Spacing** — the actual distance between consecutive brush marks while dragging.

### Mask Types

Determines the shape of the brush mask applied to the painted area.

**Custom Brushes** — use a custom texture as the brush mask. Right-click in the mask picker to access brushes already bundled with MegaWorld, or import your own textures.

**Procedural Brush** — generate a mask from mathematical parameters instead of a texture:

- **Shape** — the geometric form of the mask (circle, square, diamond, etc.).
- **Brush Falloff (%)** — controls the gradient falloff from the center to the edge of the brush.
- **Brush Strength (%)** — the maximum influence value at the center. Lower values produce softer marks.
- **Fractal Noise** — adds procedurally generated noise to the mask for an organic, irregular edge.

### Brush Jitter

Controls scatter and randomness of the brush stroke itself.

- **Brush Size** — selected prototypes only spawn within this range around the brush center.
- **Brush Scatter** — randomizes the brush position by a random offset from the stroke path.
- **Brush Rotation** — rotates the brush mask on each mark.
- **Jitter** — the overall randomness of the brush stroke.

## Shared systems used by this tool

For **Terrain Object** and **Unity/GameObject** groups, Advanced Brush uses:

- [Scatter Settings](/megaworld/scatter-system/scatter-settings)
- [Filter Settings](/megaworld/mask-filters/overview)
- [Transform Components](/megaworld/transform-components/overview)
- [Overlap Check Settings](/megaworld/overlap-check-settings/overview)
- [Additional Spawn Settings](/megaworld/general-settings/additional-spawn-settings)

For **Unity/Terrain Detail** groups, Advanced Brush uses:

- [Spawn Detail Settings](/megaworld/general-settings/spawn-detail-settings)
- mask filter settings

For **Unity/Terrain Texture** groups, Advanced Brush uses:

- mask filter settings
- **Texture Target Strength**

## When to use Advanced Brush

Advanced Brush is best when you want:

- direct scene-view painting with more control than a one-click stamper pass;
- the same filters and transform logic as the procedural tools, but with manual control over placement;
- a mixed workflow where you art-direct the placement and still keep system-driven variation.

## Notes

- Brush spacing is adjusted while dragging so dense object painting does not collapse into over-sampling.
- Different resource types expose different settings even inside the same brush workflow.
- For terrain textures, Advanced Brush paints target strength instead of spawning objects.

<Tip>
  Advanced Brush is a great default tool when you want the result to feel hand-placed but still benefit from filters, transforms, and other MegaWorld systems.
</Tip>
