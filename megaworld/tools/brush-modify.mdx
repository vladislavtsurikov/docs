Brush Modify is a Happy Artist tool for changing objects that already exist in the scene.

Instead of spawning or erasing, it applies transform modifications to overlapped objects as you drag the brush across them.

## Features

### Filter mask support

Brush Modify uses the same [Mask Filters](/megaworld/mask-filters/overview) as other MegaWorld tools. Add height, slope, or noise filters to restrict which objects get modified as you paint. Objects outside the filter range are ignored by the brush — only objects that pass the filter are affected by the transform stack.

## Supported resource types

Brush Modify supports:

- **Terrain Object**
- **Unity/GameObject**

## Tool-level controls

Brush Modify uses:

- **Brush Settings**
- **Modify Transform Components**

## Shared systems used by this tool

Brush Modify also uses:

- [Filter Settings](/megaworld/mask-filters/overview)

## Modify Transform Components

Click **+** in the Modify Transform section to add components to the stack. Components can be reordered and removed. Each component is applied in order to every overlapping object that passes the filter.

### Position

Offsets the object's position while painting.

- **Strength** — direction and magnitude of the position offset applied per stroke.
- **Strength Randomize** — how much randomness is added to the strength. `0` = no randomness, `100` = fully random offset strength.

### Vegetation Rotation

Tilts objects like trees — useful for adding natural lean and variation to vegetation.

- **Strength Y** — how much the object rotates around the Y axis.
- **Strength Rotation XY** — how quickly the effect is applied.
- **Rotation XZ** — the lean strength along Y and X axes. `1` = fully fallen tree, `0` = no lean.

### Along

Rotates objects along the Y axis in the direction of the brush stroke. Objects gradually orient themselves to the direction you are painting.

### Rotation

Applies rotation to the object's transform.

- **Strength** — how fast the rotation is applied.
- **Random Rotation (X, Y, Z)** — applies random rotation on the chosen axes.
- **Randomize (X, Y, Z)** — the randomness strength per axis.
- **Rotation (X, Y, Z)** — whether the rotation is applied in the positive or negative direction on each axis.

### Randomize Scale

Randomly reassigns the object's scale within a defined range on each stroke.

- **Uniform Scale** — applies the same scale value across all three axes.
- **Min / Max Scale** — the range within which the random scale is chosen.

### Scale

Incrementally scales the object up or down while painting.

- **Strength** — whether the scale increases (positive) or decreases (negative).
- **Strength Randomize** — the randomness added to the scale strength per stroke.

### Scale Clamp

Prevents the scale from going below or above defined limits, regardless of how many times the brush passes over an object.

## How it works

- The brush overlaps existing objects.
- The tool evaluates filter fitness and brush mask strength.
- The modify stack applies transform changes based on stroke movement and fitness.
- Changes are accumulated while you drag.

## When to use Brush Modify

Brush Modify is best when you want:

- to art-direct an already spawned area;
- to nudge, rotate, or reshape existing placement with a brush gesture;
- to refine a procedural result instead of deleting and respawning it.

| Shortcut | Action |
|---|---|
| `LShift + Scroll` | Resize brush |
