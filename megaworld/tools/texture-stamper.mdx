Texture Stamper is a stamper tool for painting **Unity Terrain Texture** layers inside a selected area.

Instead of spawning objects, it writes terrain texture data. This makes it useful for large terrain-material passes such as grass, dirt, cliffs, roadsides, or biome breakup.

<Info>
  New to Texture Stamper? Follow the step-by-step [Texture Stamper — Getting Started](/megaworld/getting-started/texture-stamper) tutorial first.
</Info>

## Supported resource types

Texture Stamper supports:

- [Unity/Terrain Texture](/megaworld/resource-types/unity-terrain-texture)

## Shared systems used by this tool

Texture Stamper uses:

- mask filter settings for each selected terrain texture prototype
- **Layer Settings** for raycast and terrain targeting
- **Stamper Tool Controller** for spawn behavior and visualization

## Area Settings

See [Area Settings](/megaworld/shared-settings/area-settings) for position, size, spawn cells, and handle configuration.

## When to use Texture Stamper

Texture Stamper is best when you want:

- a large terrain texture pass inside a controlled area;
- reusable texture stamping workflows based on a scene object;
- mask-driven texturing instead of manual terrain painting;
- cell-based processing for large texture areas.

## Notes

- Texture Stamper is focused on terrain textures only, not objects.
- The main logic comes from the mask filter stack and the selected area shape.
- If you enable spawn cells, the tool processes the area through separate bounds instead of relying on a single full-area mask.

<Tip>
  Use Texture Stamper when you want repeatable terrain texturing. Use brush-based workflows when you want more direct hand-painted control.
</Tip>
