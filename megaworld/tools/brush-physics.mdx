title: "Brush Physics"

---

Brush Physics is a PhysX Painter tool for interactive painting with physics simulation.

Instead of placing objects directly in their final pose, it spawns them into a simulated workflow so they can settle more naturally.

<Info>
  New to Brush Physics? Follow the step-by-step [Brush Physics — Getting Started](/megaworld/getting-started/brush-physics) tutorial first.
</Info>

## Supported resource types

Brush Physics supports:

- [Terrain Object](/megaworld/resource-types/terrain-object)
- [Unity/GameObject](/megaworld/resource-types/unity-gameobject)

## Tool-level controls

Brush Physics uses:

- **Brush Physics Tool Settings**
- **Physics Effects**
- **Brush Settings**

The code shows one explicit tool-specific control in **Brush Physics Tool Settings**:

- [Position Offset Y](/megaworld/shared-settings/position-offset) — shifts the spawn point above the surface before physics simulation begins.

## Shared systems used by this tool

Brush Physics uses:

- [Scatter Settings](/megaworld/scatter-system/scatter-settings)
- [Physics Transform Settings](/megaworld/shared-settings/physics-transform-settings)
- [Success](/megaworld/shared-settings/success)
- **Physics Effects**

## When to use Brush Physics

Brush Physics is best when you want:

- physics-based painting instead of direct placement;
- repeated interactive strokes with a brush workflow;
- naturally settled props, clutter, or dropped objects;
- a less rigid result than standard brush spawning.

## Notes

- Brush Physics belongs to the **PhysX Painter** family.
- It uses brush-style interaction, but the actual placement logic is physics-driven.
- The transform system here is physics-specific rather than the standard transform component stack.
