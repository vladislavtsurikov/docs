---

Precise Place is a Happy Artist tool for intentional manual placement.

It gives you a live preview object, lets you drag and reposition before confirming, and supports extra mouse-action tools for rotation, scale, and directional movement.

<Info>
  New to Precise Place? Follow the step-by-step [Precise Place — Getting Started](/megaworld/getting-started/precise-place) tutorial first.
</Info>

## Supported resource types

Precise Place supports:

- [Terrain Object](/megaworld/resource-types/terrain-object)
- [Unity/GameObject](/megaworld/resource-types/unity-gameobject)

## Tool-level controls

Precise Place uses **Precise Place Tool Settings**.

The confirmed settings include:

- **Spacing**
- **Random Select Prototype**
- **Use Transform Components**
- **Overlap Check**
- **Visualize Overlap Check Settings**
- **Enable Snap Move**
- **Snap Move**
- **Align**
- **Weight To Normal**
- **Along Stroke**

## Prototype-level controls

Precise Place also has a per-prototype block:

- **Precise Place Settings**
- currently confirmed field: **Position Offset**

## Mouse actions

The current mouse-action stack includes:

- **Move Along Direction**
- **Rotation**
- **Scale**

## Shared systems used by this tool

Precise Place uses:

- `Success`
- [Transform Components](/megaworld/transform-components/overview)
- [Overlap Check Settings](/megaworld/overlap-check-settings/overview)

## When to use Precise Place

Precise Place is best when you want:

- careful one-by-one placement with a preview object;
- manual control with optional transform and overlap assistance;
- more precision than a brush workflow, but more convenience than placing prefabs completely by hand.

<Tip>
  Use Precise Place when final art direction matters. It is especially useful for hero props, hand-authored paths, edge placements, and scene accents.
</Tip>
