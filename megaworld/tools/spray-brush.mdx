Spray Brush is a lightweight painting tool for fast object scattering in the scene.

Compared with **Advanced Brush**, this tool uses a simpler setup: simple filters, simple transform settings, and overlap checks. It is useful when you want a quick manual painting workflow without the full transform component stack.

<Info>
  New to Spray Brush? Follow the step-by-step [Spray Brush — Getting Started](/megaworld/getting-started/spray-brush) tutorial first.
</Info>

## Supported resource types

Spray Brush supports:

- [Terrain Object](/megaworld/resource-types/terrain-object)
- [Unity/GameObject](/megaworld/resource-types/unity-gameobject)

## Shared systems used by this tool

Spray Brush uses:

- [Brush Settings](/megaworld/shared-settings/brush-settings)
- [Success](/megaworld/shared-settings/success)
- [Simple Filter Settings](/megaworld/shared-settings/simple-filter-settings)
- [Simple Transform Settings](/megaworld/shared-settings/simple-transform-settings)
- [Overlap Check Settings](/megaworld/overlap-check-settings/overview)

## When to use Spray Brush

Spray Brush is best when you want:

- a very fast manual object scattering workflow;
- lighter setup than **Advanced Brush**;
- direct control with fewer moving parts;
- simple variation and overlap protection without the full advanced stack.

## Notes

- Spray Brush samples a point inside the brush radius instead of filling the whole brush area procedurally.
- The tool uses `SimpleTransformComponentSettings`, not the full [Transform Components](/megaworld/transform-components/overview) stack.
- This makes the tool quicker to set up, but also less flexible than **Advanced Brush**.

<Note>
  Use Spray Brush when speed matters more than fine-grained control. If you need terrain details, terrain textures, or the full transform stack, switch to **Advanced Brush**.
</Note>
