---
title: "Precise Place Hotkeys"
description: "Keyboard shortcuts for the Precise Place tool"
---

All hotkeys use mouse movement in the horizontal axis only.

| Shortcut | Action |
|---|---|
| `Tab` | Restore the object's transform to the original Prefab values |
| `LShift + Mouse move` | Rotate around the global (or local) Y axis |
| `LShift + X + Mouse move` | Rotate around the global (or local) X axis |
| `LShift + Z + Mouse move` | Rotate around the global (or local) Z axis |
| `LShift + F + Mouse move` | Free rotation |
| `LShift + S + Mouse move` | Screen-space rotation |
| `LCtrl + LShift + Mouse move` | Uniform scale |
| `LCtrl + LShift + X + Mouse move` | Scale on X axis |
| `LCtrl + LShift + Y + Mouse move` | Scale on Y axis |
| `LCtrl + LShift + Z + Mouse move` | Scale on Z axis |
| `Q + Mouse move` | Offset position along the surface normal (or straight up depending on settings) |
