Terrain Spawner is a stamper tool for procedural placement inside a defined area. You position and scale the stamper in the scene, then MegaWorld evaluates the selected groups and fills that volume with the prototypes that pass your rules.

This is one of the best tools when you want to generate a large area at once instead of painting manually.

## Spawn large terrain objects in one click

Terrain Spawner fills the entire stamper area in a single pass. For very large terrains, enable **Use Spawn Cells** so the area is processed cell by cell instead of all at once. This avoids long freezes and prevents Unity from crashing on extremely large areas.

## Optimized visualization for large terrains

## Spawn support for all resource types

Terrain Spawner works with every MegaWorld resource kind in a single tool:

- [Terrain Object](/megaworld/resource-types/terrain-object) and [Unity/GameObject](/megaworld/resource-types/unity-gameobject) — procedural object placement with the full shared stack
- [Unity/Terrain Detail](/megaworld/resource-types/unity-terrain-detail) — grass and mesh detail density passes

## Shared settings across tools

Types in MegaWorld are reusable presets. The same Type you configure in Terrain Spawner can be used directly in Advanced Brush. Build and tune a preset in the stamper using Auto Respawn, then hand-paint with the same rules in the brush — no duplication needed.

  style={{ borderRadius: '0.5rem', width: '100%', maxWidth: '960px' }}

## Auto Respawn

When **Auto Respawn** is on, Terrain Spawner automatically removes and re-spawns the current pass every time you change a setting. Use this during setup to get instant feedback. Set **Delay Auto Spawn** to control the respawn delay in seconds.

## Works on Unity Terrain and terrain mesh

Terrain Spawner works on both Unity Terrain and polygon terrain meshes such as [Polaris](/megaworld/polaris-integration). For mesh terrain, switch to **Simple Filter** instead of Mask Filters.

  style={{ borderRadius: '0.5rem', width: '100%', maxWidth: '960px' }}

## Cell support

Placed objects are stored in spatial cells. This enables [Overlap Check Settings](/megaworld/overlap-check-settings/overview), and makes tools like [Brush Erase](/megaworld/tools/brush-erase) and [Edit](/megaworld/tools/edit) aware of exactly where each object was placed.
