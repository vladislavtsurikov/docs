MegaWorld tools are organized by workflow rather than by asset type.

That means the first question is usually not "what am I spawning?" but "how do I want to work in the scene?"

## Main tool families

### Stamper tools

Use stamper tools when you want controlled placement over an area or a sampled set of points.

- [Terrain Spawner](/megaworld/tools/terrain-spawner) for broad terrain-aware spawning workflows
- [Texture Stamper](/megaworld/tools/texture-stamper) for painting terrain textures
- [Gravity Spawner](/megaworld/tools/gravity-spawner) for dropping objects into place with physics-oriented logic

### Brush tools

Use brush tools when you want direct scene-view painting.

- [Advanced Brush](/megaworld/tools/advanced-brush) for the most flexible paint workflow
- [Spray Brush](/megaworld/tools/spray-brush) for lighter continuous scattering
- [Brush Erase](/megaworld/tools/brush-erase) for removing placed content

### PhysX Painter tools

Use these when physical interaction is part of the workflow.

- [Brush Physics](/megaworld/tools/brush-physics)
- [Precise Physics](/megaworld/tools/precise-physics)
- [Explode Physics](/megaworld/tools/explode-physics)
- [Physics Effects](/megaworld/tools/physics-effects)

### Happy Artist tools

Use these when hand-editing and finishing matter more than mass placement.

- [Brush Modify](/megaworld/tools/brush-modify)
- [Edit](/megaworld/tools/edit)
- [Precise Place](/megaworld/tools/precise-place)
- [Pin](/megaworld/tools/pin)

## Shared systems across tools

Many tools reuse the same support systems:

- [Mask Filters](/megaworld/mask-filters/overview)
- [Transform Components](/megaworld/transform-components/overview)
- [Overlap Check Settings](/megaworld/overlap-check-settings/overview)
- [Scatter System](/megaworld/scatter-system/overview)
- [General Settings](/megaworld/general-settings/overview)

## How to choose quickly

- Choose a **stamper** when you want area-based generation.
- Choose a **brush** when you want direct painting.
- Choose a **physics** tool when object interaction matters.
- Choose a **Happy Artist** tool when you want manual polish and cleanup.

<Tip>
  If you are unsure where to start, begin with [Terrain Spawner](/megaworld/tools/terrain-spawner) for broad placement or [Advanced Brush](/megaworld/tools/advanced-brush) for hands-on painting.
</Tip>
