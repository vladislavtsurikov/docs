---
title: "Edit Hotkeys"
description: "Keyboard shortcuts for the Edit tool"
---

| Shortcut | Action |
|---|---|
| `Q` | Switch active action to Move Up/Down |
| `W` | Switch active action to Raycast |
| `E` | Switch active action to Rotate |
| `R` | Switch active action to Scale |
| `T` | Switch active action to Remove |
| `Space` | Toggle transform space (global / local) — applies to Rotate |

For Move Up/Down, Rotate, and Scale: after clicking a handle button, hold the mouse button and drag horizontally to apply the change.
