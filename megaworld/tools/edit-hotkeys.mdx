---
title: "Edit Hotkeys"
description: "Keyboard shortcuts for the Edit tool"
---

Hotkeys differ depending on whether **Unity Handles** or **Mouse Modify** mode is active.

## Mouse Modify mode

| Shortcut | Action |
|---|---|
| `W` | Position |
| `Q` | Move Along Direction |
| `Ctrl + LShift` | Raycast position |
| `E` | Rotation |
| `Space` | Toggle transform space (global / local) |
| `R` | Scale |
| `T` | Remove |

## Unity Handles mode

| Shortcut | Action |
|---|---|
| `W` | Position |
| `E` | Rotation |
| `R` | Scale |
| `T` | Remove |
| `Space` | Toggle transform space (global / local) |
