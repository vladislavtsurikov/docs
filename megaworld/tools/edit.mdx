Edit is a Happy Artist tool for direct object manipulation.

It is built around an action stack rather than a brush or stamper workflow, which makes it suitable for targeted edits to already placed objects.

<Info>
  New to Edit? Follow the step-by-step [Edit — Getting Started](/megaworld/getting-started/edit) tutorial first.
</Info>

## Features

### Unity Handles

Edit supports standard Unity transform gizmos (position, rotation, scale). Switch to Unity Handles mode when you need precise control using the familiar Unity viewport workflow.

  style={{ borderRadius: '0.5rem', width: '100%', maxWidth: '960px' }}

### Mouse Modify

Edit also supports a direct mouse-driven modification mode. In Mouse Modify mode you can move, rotate, scale, and raycast-align objects by holding hotkeys and dragging the mouse — without clicking through separate gizmo handles. This is faster for iterative adjustments.

### Remove

Edit can delete individual placed objects directly in the scene view. This lets you clean up placement without switching tools or using the hierarchy.

## Supported resource types

Edit supports:

- [Terrain Object](/megaworld/resource-types/terrain-object)
- [Unity/GameObject](/megaworld/resource-types/unity-gameobject)

## Tool-level controls

Edit uses **Edit Tool Settings** with the following fields:

- **Sphere Size** — the radius around the raycast hit point within which objects are detected and made eligible for editing.
- **Max Distance** — the maximum distance from the camera at which objects can be selected for modification.
- **Ground Layers** — the layers that the raycast hits to detect ground and locate objects around the impact point.
- **Mouse Rotation Sensitivity** — how strongly mouse movement translates into transform changes in Mouse Modify mode.
- **Transform Space** — switches between local and global transform space for all actions.
- **Raycast Position Offset** — adds a Y-axis offset to the raycast position result, useful for correcting ground-level alignment.

## Available actions

| Action | What it does |
|---|---|
| **Move Up / Down** | Shifts the object along its local Y axis |
| **Raycast** | Snaps the object down onto the terrain surface using a raycast |
| **Rotate** | Rotates the object |
| **Scale** | Scales the object |
| **Remove** | Deletes the object from the scene |

## When to use Edit

Edit is best when you want:

- direct targeted edits instead of broad brush operations;
- to switch between edit actions quickly;
- precise cleanup and object adjustment after placement.
