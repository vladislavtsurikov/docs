The MegaWorld window is the main entry point for the whole tool ecosystem.

Open it from `Window > Vladislav Tsurikov > Mega World > Window`.

If you want a modular layout, open `Window > Vladislav Tsurikov > Mega World > Separate Windows`.

## Shortcuts

- `Alt + Z` opens the main MegaWorld window.
- `Alt + X` opens the separate windows layout.

The interface is built around three core panels:

- [Tools panel](/megaworld/tools-panel) for choosing the active tool.
- [Selection panel](/megaworld/selection-panel) for choosing resource types, types, and prototypes.
- [Settings panel](/megaworld/settings-panel) for editing the settings of the selected tool and the selected prototype.

## How the workflow flows through the window

In practice, you move through the window in this order:

1. Choose the tool.
2. Choose what the tool should work with.
3. Configure the settings for that tool and those prototypes.
4. Use the scene view to paint, stamp, place, erase, or edit.

This structure stays consistent across most MegaWorld workflows, even when the actual settings differ between tools.
