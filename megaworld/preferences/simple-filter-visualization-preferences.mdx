Simple Filter Visualization Preferences control how MegaWorld previews simple-filter results in the scene view.

<Warning>
  This is an advanced user setting. The defaults work well in most projects. Adjust only if the visualization is hard to read in your specific scene.
</Warning>

<Tip>
  If rendering slows down while painting, disable **Enable Spawn Visualization** here. This gives a noticeable performance improvement in complex scenes.
</Tip>

## Settings

### Enable Spawn Visualization

**Default:** `true`

Master toggle for all simple-filter spawn visualization. When off, MegaWorld skips drawing the preview entirely, which improves performance.

### Visualiser Resolution

**Default:** `15`  
**Range:** `1–60`

Controls how dense the preview sampling grid is. Higher values draw more handles and give a finer preview but cost more to render. Lower values are faster but may miss detail.

### Handles Type

**Default:** `DotCap`

Shape of the handles drawn in the preview:

- **DotCap** — flat dot handles, lightweight and easy to read at a distance
- **Sphere** — 3D sphere handles, more visible in perspective but heavier

### Handle Resizing Type

**Default:** `Distance`

How handle size scales as you move the camera:

- **Distance** — handles scale with camera distance so they stay a consistent apparent size
- **Resolution** — handle size follows the visualiser resolution setting
- **CustomSize** — uses the value from **Custom Handle Size**

### Custom Handle Size

**Default:** `0.3`  
**Range:** `0.1–3.0`

Visible only when **Handle Resizing Type** is set to `CustomSize`. Sets a fixed world-space size for all handles.

### Color Handles Type

**Default:** `Standard`

- **Standard** — uses MegaWorld's built-in active/inactive colors
- **Custom** — lets you define your own active and inactive colors

### Active Color / Inactive Color

Visible only when **Color Handles Type** is `Custom`. Set the colors used for areas that pass or fail the filter.

### Alpha

**Default:** `0.6`  
**Range:** `0–1`

Overall opacity of the preview visualization.

## Effect on Simple Filter Settings

These preferences control the look of the preview drawn by [Simple Filter Settings](/megaworld/shared-settings/simple-filter-settings). They do not affect filter logic or placement results.
