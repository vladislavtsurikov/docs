---
title: "Brush Settings"
description: "Control brush size, spacing, mask shape, and brush interaction behavior across brush-based MegaWorld tools"
---

Brush Settings define the shape and behavior of scene-view brush interaction in MegaWorld.

Different tools use different brush variants, but they all build on the same core idea: brush size, brush spacing, and a mask that defines the affected area.

## Core settings

The shared base layer includes:

- **Brush Size**
- **Spacing**
- brush radius derived from brush size
- brush mask output used by the current tool

## Interaction shortcut

You can resize the brush with:

- `Shift + Mouse Wheel`

## Advanced brush variant

Some tools, such as **Advanced Brush** and **Brush Erase**, use an extended brush settings block that adds:

- brush jitter
- custom masks
- procedural masks
- spacing modes such as brush-size-based spacing
- brush rotation

## Preferences

The brush system also has preference-level settings such as:

- [Brush Preferences](/megaworld/preferences/brush-preferences)
- [Brush Visualization Preferences](/megaworld/preferences/brush-visualization-preferences)

## Where Brush Settings are used

Brush Settings are central to tools such as:

- [Advanced Brush](/megaworld/tools/advanced-brush)
- [Spray Brush](/megaworld/tools/spray-brush)
- [Brush Erase](/megaworld/tools/brush-erase)
- [Brush Physics](/megaworld/tools/brush-physics)
- [Brush Modify](/megaworld/tools/brush-modify)

<Note>
  If a brush tool feels too dense or too sparse, check **Brush Size** and **Spacing** before changing the deeper placement logic.
</Note>
