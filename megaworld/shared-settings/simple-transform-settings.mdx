---
title: "Simple Transform Settings"
description: "A lightweight transform stack for tools that do not use the full Transform Components workflow"
---

Simple Transform Settings provide a lighter transform stack than the full [Transform Components](/megaworld/transform-components/overview) system.

This block is used by tools that need quick transform variation without exposing the full advanced transform workflow.

## Default components

The current code creates these components by default:

- **Position Offset**
- **Rotation**
- **Scale**

## When to use it

Simple Transform Settings are most relevant in workflows such as:

- [Spray Brush](/megaworld/tools/spray-brush)

## Difference from full Transform Components

- Simple Transform Settings are more lightweight.
- The full Transform Components system supports a broader modular stack and more specialized components.
- Use the full system when you need slope-aware or cliff-specific behavior.
