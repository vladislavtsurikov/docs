---
title: "Spacing"
description: "Set the minimum distance between consecutive placements in MegaWorld tools"
---

Spacing sets the minimum distance between consecutive placement actions.

When you move the cursor across the scene and interact with a tool, MegaWorld checks whether the cursor has moved far enough from the last placement point before spawning the next object. If the distance is less than the Spacing value, the action is skipped.

This prevents objects from stacking on top of each other during continuous interaction.

- A lower value places objects more densely.
- A higher value forces more distance between each placement.

## Where it is used

- [Precise Physics](/megaworld/tools/precise-physics)
- [Explode Physics](/megaworld/tools/explode-physics)
- [Precise Place](/megaworld/tools/precise-place)

<Note>
  For brush-based tools, spacing is controlled inside [Brush Settings](/megaworld/shared-settings/brush-settings) under the Brush Spacing section.
</Note>
