Stamper Tool Controller is the main action and mode panel for stamper-based tools.

It controls visualization, auto respawn behavior, and the core spawn actions.

## Settings

**Visualisation** — shows the Mask Filter output directly in the scene view so you can see which areas pass your filters before spawning. Enabled by default. Visualization appearance is configured in [Preferences](/megaworld/preferences/overview).

**Auto Respawn** — when enabled, MegaWorld automatically removes and re-spawns the current pass every time you change a setting. Gives instant live feedback while tuning.

- **Delay Auto Spawn** — how many seconds to wait after a setting change before the automatic respawn triggers. Range is 0–3 seconds. Default is `0.1`.

## Buttons

When **Auto Respawn** is on:

- **Respawn** — removes all spawned instances for all groups and runs a fresh spawn pass.

When **Auto Respawn** is off:

- **Respawn** — removes all spawned instances for all groups and runs a fresh spawn pass.
- **Spawn** — adds a new spawn pass on top of whatever is already in the scene without removing previous instances.
- **Cancel** — cancels a spawn pass that is currently in progress.

Always visible:

- **Unspawn Selected Prototypes** — removes only the instances belonging to the currently selected prototypes. Shows a confirmation dialog before proceeding.
