These are the core hotkeys currently confirmed in the MegaWorld code and the original documentation.

## Window shortcuts

- `Alt + Z` opens the main MegaWorld window.
- `Alt + X` opens the separate windows layout.

## Workflow shortcut

- `Ctrl + Backspace` removes all spawned data from the selected prototypes.

## Notes

- The exact effect of a shortcut can depend on the selected tool and the selected prototype type.
- When you use destructive actions such as removing spawned data, make sure the correct prototypes are selected first.
