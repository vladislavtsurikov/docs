The Tools panel lets you choose which MegaWorld tool you want to work with.

<img
  style={{ borderRadius: '0.5rem', width: '100%', maxWidth: '522px' }}
  src="/images/megaworld/window-workflow/tools-panel.png"
  alt="MegaWorld Tools panel."
/>

## What you do here

- Select the active tool.
- Reorder tools to match your workflow.
- Remove tools you do not need.
- Add tools back when you want them again.

## Tool families

MegaWorld tools are grouped by workflow:

- **Stamper tools** such as **Terrain Spawner**, **Texture Stamper**, and **Gravity Spawner**.
- **Brush tools** such as **Advanced Brush**, **Spray Brush**, and **Brush Erase**.
- **PhysX Painter** tools such as **Brush Physics**, **Precise Physics**, **Explode Physics**, and **Physics Effects**.
- **Happy Artist** tools such as **Brush Modify**, **Edit**, **Precise Place**, and **Pin**.

## Common actions

- Right-click a tool to remove it from the current layout.
- Reorder tools when you want frequently used tools closer together.
- Use the separate windows layout if you want the tool list in its own window.
