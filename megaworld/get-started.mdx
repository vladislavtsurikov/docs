MegaWorld is a multi-tool ecosystem, so the easiest way to get started is to begin with one practical workflow instead of trying to learn every tool at once.

For your first pass, use **Terrain Spawner** or **Advanced Brush** to build a simple vegetation setup.

## 1. Open MegaWorld

Open the main window from `Window > Vladislav Tsurikov > Mega World > Window`.

If you prefer a split layout, use `Window > Vladislav Tsurikov > Mega World > Separate Windows`.

## 2. Pick a first workflow

Start with one of these:

- **Terrain Spawner** when you want to fill an area procedurally.
- **Advanced Brush** when you want to paint interactively in the scene.
- **Texture Stamper** when you want to stamp terrain textures instead of objects.

## 3. Select prototypes

In the selection area, choose the resource type you want to work with and add the prototypes you want to spawn.

Common starting points:

- **Unity/GameObject** for rocks, trees, props, and prefabs.
- **Unity/Terrain Detail** for grass and terrain details.
- **Unity/Terrain Texture** for terrain painting workflows.

## 4. Add the core shared systems

Most useful setups need at least some of these:

- [Mask Filters](/megaworld/mask-filters/overview) to restrict placement by terrain data.
- [Transform Components](/megaworld/transform-components/overview) to make objects feel natural instead of repeated.
- [Overlap Check Settings](/megaworld/overlap-check-settings/overview) to stop intersections and clipping.

## 5. Build a simple first setup

A good first example is a forest pass:

1. Add tree prefabs as prototypes.
2. Use a height and slope-based filter setup.
3. Add transform components for rotation and scale variation.
4. Enable overlap checks so trunks do not intersect.
5. Stamp or paint the result into the scene.

<Tip>
  Do not try to configure everything at once. Start with placement first, then add variation, and only after that refine overlap, scatter, and visualization.
</Tip>
